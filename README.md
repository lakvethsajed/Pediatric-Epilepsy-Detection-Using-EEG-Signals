Dataset

This project uses the CHB-MIT Scalp EEG Dataset for pediatric epilepsy detection.

Original Dataset (PhysioNet):
https://physionet.org/content/chbmit/1.0.0/#files-panel
Preprocessed EEG Dataset (from Google Drive):
https://drive.google.com/drive/folders/1Mn9Dsf4CkZn4g3TVEmjVNh4g9RxogywJ
GAF Image Dataset (Converted from EDF to PNG):
https://drive.google.com/drive/folders/1IlMD8ZUBmtEpPyi8IJN4f49luLzKUucr

⚠️ Notes
The original dataset is provided by PhysioNet and requires proper citation for academic use.
The GAF dataset contains EEG signals transformed into image representations for deep learning.
Due to size constraints, datasets are hosted externally via Google Drive.
